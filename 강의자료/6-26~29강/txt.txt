static 메서드 - 인스턴스 생성 X, '클래스이름.메서드이름'
인스턴스 메서드 - 인스턴스 생성 O, '참조변수.메서드이름'

차이점 
 - iv 사용 여부
 - static이 앞에 붙느냐
 - 인스턴스 메소드는 객체를 생성해야 사용 가능하고, static메소드는 객체를 생성하지 않아도 사용이 가능하다

static 변수랑 똑같다고 생각하면 됨


언제 static을 붙어야 할까 ?
 - 속성(멤버 변수) 중에서 공통 속성에 static을 붙인다
 - 인스턴스 멤버(iv)를 사용하지 않는 메서드에 static을 붙인다

호출과 참조
 - static 메서드는 static메서드 호출 가능
 - static 메서드는 인스턴수 변수 사용 불가능
 - static 메서드는 인스턴스 메서드 호출 불가능
 - static 메서드는 호출시 객체가 없을 수도 있어서 인스턴스 멤버 사용 불가능
